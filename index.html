§ 03 / FLOW · 一次完整的“做事”

从一句话，
到事情真的被做完。 From one sentence — to the task actually done

STEP 01

说人话

Say it in plain words

直接用中文或英文描述你想做的事：整理、记录、查询、推送、长期追踪 —— 不用想工具名字。

STEP 02

Agent 结合仓库、笔记、资源和已装技能，决定用哪些工具：读文件、搜索、写笔记、发通知……

STEP 03

动手执行

Tools actually run

文件真的被读写、终端命令真的被执行、搜索结果真的被抓取，全过程在工作台里可见、可撤销。

STEP 04

持续跟进

Keeps running for you

定时任务、循环任务、微信回推 —— 即使你不在电脑前，Agent 也会继续把事跟到底。

§ 06 / FAQ · 常见问题

那些可能先想问的事。 Things you'll probably ask first

Q 01 我的数据会被上传吗？+
不会。1052 OS 是本地优先的：文件、笔记、资源、长期记忆、定时任务全部保存在你自己的机器上。 只有当你选择调用云端模型或联网搜索时，相关内容才会按需发送给你配置的服务。 Local-first. Your data stays on your machine unless you opt in to a cloud model or web search.
Q 02 支持哪些模型？+
任意 OpenAI 兼容接口都能直接配置，包括本地模型（如 Ollama / vLLM 暴露的兼容端点）。 你可以为不同场景配置不同模型：聊天用一个、定时任务用另一个。 Any OpenAI-compatible endpoint works — including local ones. Per-scenario model config is supported.
Q 03 它和普通 AI 聊天 App 有什么不同？+
普通 App 给你一个聊天框；1052 OS 给你一个会记东西、会动手、会守夜的工作台： 本地仓库、笔记、技能、定时任务和社交通道都被编织在同一个 Agent 周围。 Chat apps give you a box. 1052 OS gives you a workspace — one that remembers, executes, and keeps running.
Q 04 定时任务和「长期任务」有什么区别？+
单次任务在指定时间执行一次；循环任务按周期重复；长期任务是没有固定终点的目标型任务 —— 例如"帮我追一个项目直到上线"，Agent 会持续读取相关资源并在有进展时通知你。 Once / recurring / long-term — long-term tasks are open-ended goals the agent keeps tracking.
Q 05 微信通道安全吗？+
微信通道通过扫码接入，消息通路在本地代理；你可以随时在设置里撤销授权。 任何通过社交通道发出的定时任务都会在通知中心留下可追溯的记录。 Scan-to-link, revokable anytime. Every outgoing action leaves an auditable trace in notifications.
Q 06 开源吗？可以自己写技能吗？+
项目采用 MIT 许可。技能中心同时支持本地技能和技能市场，你可以编写、安装、预览并热更新技能， 也可以把技能打包分享给别人。 MIT licensed. Write your own skills locally, or install & hot-reload from the skill market.
