## 核心模块

### 1. 聊天引擎（core/chat_engine.py）

- 处理用户消息
- 调用 LLM API（OpenAI）
- 流式输出响应
- 工具调用协调

### 2. 配置管理（core/config.py）

- 加载配置文件
- 环境变量支持
- 配置持久化

### 3. 技能管理器（core/skill_manager.py）

- 动态加载技能插件
- 热重载支持
- 技能注册和调用

### 4. 任务调度器（core/scheduler.py）

- Cron 表达式解析
- 定时任务执行
- 任务持久化

### 5. IM 集成管理器（im_integration/manager.py）

- 统一管理多个 IM 平台
- 消息路由
- 平台适配

## 数据流

### 聊天流程

```
用户输入 → API 路由 → 聊天引擎 → LLM API
                                    ↓
                              工具调用（可选）
                                    ↓
                              流式响应 → 用户
```

### 定时任务流程

```
调度器 → 检查任务 → 执行任务 → 调用聊天引擎 → 发送结果
```

## 安全机制

1. **API 认证**：可选的 API Key 认证
2. **CORS 限制**：仅允许本地访问
3. **环境变量**：敏感信息不存储在代码中
4. **日志记录**：完整的操作日志

## 扩展性

### 添加新的 IM 平台

1. 在 `im_integration/` 创建新的 bot 文件
2. 实现统一的消息接口
3. 在 `manager.py` 中注册

### 添加新的工具

1. 在 `core/tools.py` 添加工具函数
2. 注册到工具列表
3. LLM 自动调用

### 添加新的技能

1. 在 `skills/` 目录创建 Python 文件
2. 实现技能逻辑
3. 自动热加载

## 性能考虑

- 流式输出减少延迟
- 异步 I/O 提高并发
- 日志文件自动轮转
- 对话历史限制大小

## 监控和告警

- 健康检查端点（/health）
- 后台监控任务
- 告警日志记录
- 服务状态追踪
